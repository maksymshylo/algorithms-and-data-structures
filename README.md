# Algorithms and Data Structures

Labs from the Institute of Physics and Technology, KPI.

This repository contains small educational implementations of common data structures and algorithms in **Python** and **C++**.

## Repository Structure

- `lab1/` — simple algorithms and recursion
- `lab2/` — sorting algorithms
- `lab3/` — arrays and linked lists in C++
- `lab4/` — stacks, queues, and linked lists
- `lab5/` — binary search tree practice

## Topics Covered

- recursion
- binary search
- factorial and Fibonacci
- Euclidean algorithm
- stacks
- queues
- singly, doubly, and circular linked lists
- binary search trees
- tree traversal and deletion

## How to Run

Most examples are standalone scripts.

### Python
Run a file with:
```bash
bash python filename.py
``` 

### C++
Compile and run a file with:
```bash
bash g++ filename.cpp -o program ./program
```
